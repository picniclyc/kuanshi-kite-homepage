# 款世kite官网

款世 Kite 官网公开展示仓。

这个仓库只负责一件事：承接对外可直接打开的官网静态页面，用于 GitHub Pages 展示和外部分享。

## 在线访问

- 官网预览：[打开款世 Kite 官网](https://picniclyc.github.io/kuanshi-kite-homepage/)
- 仓库地址：[picniclyc/kuanshi-kite-homepage](https://github.com/picniclyc/kuanshi-kite-homepage)

## 当前定位

- 这是公开品牌展示仓
- 内容为静态首页快照
- 适合直接分享和对外查看

## 仓库结构

```text
kuanshi-kite-homepage/
├── index.html
├── 404.html
├── favicon.ico
├── _next/
├── fonts/
├── images/
├── media/
└── .nojekyll
```

## 说明

- 私有源码仓继续保留在 `kite-homepage-final`
- 这个公开仓不承担 Next.js 开发职责
- 如果首页有新版本，建议从源码仓重新构建后覆盖这里的静态文件
